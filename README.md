Итоговый проект по базам данных Михеевой Софьи.

Сама база данных называется library и имеет в себе три таблицы - Book, Reader и Issuance(выдача)
Структуру таблиц можно посмотреть в models.py

Основа таблиц (все, что было в билете с 1 модуля) была создана с помощью create_all() из sqlalchemy, а все остальное (две миграции для 4 пункта и еще две для 6 пункта) - с помощью миграций alembic.
